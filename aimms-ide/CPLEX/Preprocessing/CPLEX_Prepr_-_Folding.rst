Folding
=======



:Type:	Selection	
:Range:	The settings listed below	
:Default:	Automatic	



This option determines whether folding will be automatically executed, during the preprocessing phase, in a LP model. Possible values are:



*	Automatic
*	Off
*	Moderate
*	Aggressive
*	Very aggressive
*	Highly aggressive
*	Extremely aggressive
