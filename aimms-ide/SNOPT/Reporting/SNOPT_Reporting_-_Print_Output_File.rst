Print Output File
=================



**Type**:	Selection	

**Range**:	The settings listed below	

**Default**:	Off	



If this option is turned on, a file with the name 'snopt.prn' is created (in the log-directory). In this file a detailed iteration log and error messages are printed.



Possible values are:



*	Off
*	On
