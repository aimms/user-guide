Renaming a Font
===============

To rename a font:

1.	While in Edit Mode, select an object.

2.	Open the Properties |img_def_Properties_button_bmp| dialog box.

3.	Select the Font tab.

4.	Select the font name you wish to rename.

5.	Click Rename.

6.	Enter a new font name.

7.	Click Ok.
