

.. _option-AIMMS-put_translate_inf:


Put Translate Inf
=================



:Type:	Floating point number	
:Range:	(-inf,inf)	
:Default:	1e150	



This option specifies to which number inf is translated by the PUT statement if the option **Put Translate Special Values** is set equal to 'On'.
