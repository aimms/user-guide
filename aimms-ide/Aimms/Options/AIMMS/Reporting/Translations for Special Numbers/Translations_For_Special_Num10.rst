

.. _option-AIMMS-put_translate_undf:


Put Translate UNDF
==================



:Type:	Floating point number	
:Range:	(-inf,inf)	
:Default:	0	



This option specifies to which number undf is translated by the PUT statement if the option **Put Translate Special Values** is set equal to 'On'.
