Save Constraint Relaxation Only
===============================



:Type:		Selection	
:Range:	The settings listed below	
:Default:	On	



This option determines whether the relaxation only values of the constraints are saved in a case or not. Possible values are:

    *	Off (do not save)
    *	On (save)
