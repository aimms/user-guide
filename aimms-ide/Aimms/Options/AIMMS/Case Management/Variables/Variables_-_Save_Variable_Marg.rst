Save Variable Marginals
=======================



**Type**:	Selection	

**Range**:	The settings listed below	

**Default**:	On	



This option determines whether the marginal values of the variables as provided by the solver are saved in a case or not. Possible values are:



*	Off (do not save)
*	On (save)
