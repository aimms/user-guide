XML Number Width
================



**Type**:	Integer	

**Range**:	{-1..32}	

**Default**:	0	



This option specifies the minimal width of the string used to represent a number exported with XML. If necessary the string will start with spaces. If this options has the value 0 the width of the string will be so that it exactly contains the number, taking into account the setting of the option XML Number Precision.
