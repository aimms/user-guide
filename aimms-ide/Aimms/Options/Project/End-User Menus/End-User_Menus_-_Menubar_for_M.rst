Menubar for Message Window
==========================



:Type:	Name of a menubar	
:Range:	AIMMS menubar	
:Default:	\-	



With this option you can specify an alternative menubar for the :ref:`aimmshelp13-Messages_Window`.



**Note** 

*	This menubar will only be used in the end-user version of AIMMS.
