Menubar for Progress Window
===========================



:Type:	Name of a menubar	
:Range:	AIMMS menubar	
:Default:	\-	



With this option you can specify an alternative menubar for the progress window.



**Note** 

*	This menubar will only be used in the end-user version of AIMMS.
