Solver Listing Messages
=======================



:Type:	Selection	
:Range:	The settings listed below	
:Default:	Never	



This option determines if and which messages of the solver are included in the solver listing file. Possible values are:



*	Never
*	Error
*	Warning
*	Remark
*	All
