As-Server
=========



:Long Name:	\--as-server	
:Short Name:	<none>	
:Argument:	<none>	

The \--server option extends the \--hidden option, and should be used when AIMMS is started with limited priviliges by a system service
(e.g. through the Internet Information Server).
