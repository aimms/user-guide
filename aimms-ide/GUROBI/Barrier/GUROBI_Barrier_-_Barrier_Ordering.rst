Barrier Ordering
================



:Type:	Selection	
:Range:	The settings listed below	
:Default:	Automatic	



This option chooses the barrier sparse matrix fill-reducing algorithm. At the default value, Gurobi chooses automatically. Possible values are:



    *	Automatic
    *	Approximate minimum degree
    *	Nested dissection
