Cut Aggregation Passes Limit
============================



**Type** :	Integer	

**Range** :	{-1 .. 2000000000}	

**Default** :	-1	



This option controls the maximum number of constraint aggregation passes performed during cut generation. At the default setting of -1, Gurobi decides.
