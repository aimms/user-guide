Pricing
=======



:Type:	Selection	
:Range:	The settings listed below	
:Default:	Automatic	



This option determines the simplex variable pricing strategy. Possible values are:



    *	Automatic
    *	Partial pricing
    *	Steepest edge
    *	Devex
    *	Quick-start steepest edge
