Crossover
=========



**Type** :	Selection	

**Range** :	The settings listed below	

**Default** :	Automatic	



This option specifies whether to use crossover. Possible values are:



*	Automatic
*	Off
*	On




Setting 'Automatic' implies: only run crossover when the LP solution is not primal-dual feasible..
