Rounding Heuristic Level
========================



**Type**:	Selection	

**Range**:	The settings listed below	

**Default**:	Automatic	



This option specifies the level of rounding heuristics. Possible values are:



*	Automatic
*	Off
*	Fast
*	Normal
*	Aggressive
