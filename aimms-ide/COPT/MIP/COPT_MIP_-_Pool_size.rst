.. _option-COPT-pool_size:


Pool Size
=========



:Type:	Integer	
:Range:	{0 .. 2000000000}	
:Default:	0	



This option determines how many MIP solutions are stored in the :ref:`COPT_Solution_Pool` .



**Note** 

*	This option only affects mixed integer programming (MIP) models.

*	:ref:`COPT_Solution_Pool`
