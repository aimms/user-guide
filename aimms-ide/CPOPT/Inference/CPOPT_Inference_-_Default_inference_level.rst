Default inference level
=======================



:Type:	Selection	
:Range:	The settings listed below	
:Default:	Basic	



This option specifies the general inference level for constraints whose particular inference level is 'Default'. Possible values are:



*	Low
*	Basic
*	Medium
*	Extended
