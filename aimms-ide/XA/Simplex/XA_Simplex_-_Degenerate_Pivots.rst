Degenerate Pivots
=================



:Type:	Integer	
:Range:	{0..2100000000}	
:Default:	0	



This option defines the number of consecutive degenerate pivots before anti-cycling code is activated. In the default setting of 0 this number equals the square root of the number of rows.
