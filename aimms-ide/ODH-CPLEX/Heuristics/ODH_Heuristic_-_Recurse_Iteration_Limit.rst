Recurse Iteration Limit
=======================



:Type:	Integer	
:Range:	{0 .. 2100000000}	
:Default:	40	



This option specifies the iteration limit for the recurse heuristic for sub-solves.



This option is only active if the option **Recurse**  is set to a value other than 'Off'.
