ODH Presolve
============



**Type**:	Selection	

**Range**:	The settings listed below	

**Default**:	Yes	



This option is an indicator for the ODH engine using a separate presolve within ODH-CPLEX. Possible values are:



*	No (do not do a separate presolve)
*	Yes (do a separate presolve)
