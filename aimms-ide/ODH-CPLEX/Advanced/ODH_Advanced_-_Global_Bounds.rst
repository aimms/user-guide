Global Bounds
=============



**Type** :	Integer	

**Range** :	{0 .. 5}	

**Default** :	2



This option specifies the usage of global bounds from CPLEX caller. Possible values are:




.. list-table::

   * - 0
     - Never use
   * - 1-4
     - Intensity of use (default: 2)
   * - 5
     - Always use
