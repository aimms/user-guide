Branch
======



**Type** :	Selection	

**Range** :	The settings listed below	

**Default** :	Algorithm decides	



This option is used to decide which branch, the up branch or the down branch, should be taken first at each node. Possible values are:



*	Down branch first
*	Algorithm decides
*	Up branch first
