Create Basis from Starting Point
================================



:Type:	Selection	
:Range:	The settings listed below	
:Default:	Off	



It this option is turned on, AIMMS will try to create a basis from the starting point. This basis will then be used by MINOS.



Possible values are:



*	Off
*	On
