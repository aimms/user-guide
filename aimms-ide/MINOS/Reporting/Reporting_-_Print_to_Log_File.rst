Print to Log File
=================



**Type** :	Selection	

**Range** :	The settings listed below	

**Default** :	Off	



If this option is turned on, a file with the name 'MINOS.PRN' is created. In this file a detailed iteration log, error messages and the final solution status are printed.



Possible values are:



*	Off
*	On
