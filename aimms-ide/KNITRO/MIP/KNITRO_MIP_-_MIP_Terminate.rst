MIP Terminate
=============



**Type**:	Selection	

**Range**:	The settings listed below	

**Default**:	Optimal	



This option specifies the condition for terminating the MIP algorithm. Possible values are:



*	Optimal
*	Feasible




Setting 'Feasible' means termination at the first integer feasible point.
