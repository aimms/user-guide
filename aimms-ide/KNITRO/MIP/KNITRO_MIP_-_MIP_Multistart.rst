MIP Multistart
==============



:Type:	Selection	
:Range:	The settings listed below	
:Default:	Off	



This option specifies the use of MIP multi-start at the branch-and-bound level. Possible values are:



*	Off
*	On




Setting 'On' enables MIP multi-start for branch-and-bound.
