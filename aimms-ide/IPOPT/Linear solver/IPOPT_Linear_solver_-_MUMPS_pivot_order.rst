MUMPS pivot order
=================



:Type:	Integer	
:Range:	{0..7}	
:Default:	7	



Controls pivot order in MUMPS. (This is ICNTL(7) in MUMPS.)



**Note** 

*	This option is only used if the option **Linear Solver Selection**  has been set to 'MUMPS solver'.
