MUMPS scaling
=============



:Type:	Integer	
:Range:	{-2..7}	
:Default:	7	



This option controls scaling in MUMPS. (This is ICNTL(8) in MUMPS.)



**Note** 

*	This option is only used if the option **Linear Solver Selection**  has been set to 'MUMPS solver'.
