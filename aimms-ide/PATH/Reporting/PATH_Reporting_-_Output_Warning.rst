Output Warnings
===============



**Type**:	Selection	

**Range**:	The settings listed below	

**Default**:	No	



If the value of this option is set to 'Yes' and the option **Create Status File**  is activated, PATH writes warning messages to the status file. Possible values are:



*	No
*	Yes
