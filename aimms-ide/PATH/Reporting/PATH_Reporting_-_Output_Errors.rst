Output Errors
=============

 

**Type**:	Selection	

**Range**:	The settings listed below 	

**Default**:	Yes	



If the value of this option is set to 'Yes' (default) and the option **Create Status File**  is activated, PATH writes error messages to the status file. Possible values are:



*	No
*	Yes
