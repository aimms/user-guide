Minimal Problem Dimension
=========================



**Type** :	Integer	

**Range** :	{1 .. 1000000}	

**Default** :	1	



If the dimension of the problem is smaller than the value of this option, PATH does not use the ("Pnewton') crash method before actually solving the problem.
