Crash Iteration Limit
=====================



**Type** :	Integer	

**Range** :	{0 .. 1000000}	

**Default** :	50	



This option sets the maximal number of iterations allowed in the crash phase. This option is ignored if the value of the option **Crash Method**  is 'None'.
