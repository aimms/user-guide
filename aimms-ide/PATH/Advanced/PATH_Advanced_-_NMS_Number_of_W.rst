NMS Number of Watchdog Steps
============================



:Type:	Integer	
:Range:	{0 .. 1000000}	
:Default:	5	



This option determines the maximal number of so-called 'watchdog steps' that are allowed.



**Remark** 

The setting of this option should only be changed by specialist users.
