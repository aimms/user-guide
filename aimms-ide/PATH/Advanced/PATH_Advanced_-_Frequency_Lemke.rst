Frequency of Lemke Starts
=========================



**Type** :	Selection	

**Range** :	The settings listed below	

**Default** :	Automatic	



The frequency of Lemke starts. Possible values are:



*	Automatic
*	First
*	Always




**Remark** 


The setting of this option should only be changed by specialist users.
