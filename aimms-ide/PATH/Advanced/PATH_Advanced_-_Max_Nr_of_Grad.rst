Maximal Number of Gradient Steps
================================



**Type** :	Integer	

**Range** :	{0 .. 1000000}	

**Default** :	5	



The maximal number of gradient steps allowed before restarting.



**Remark** 

The setting of this option should only be changed by specialist users.
