NMS Merit Function Values Storage
=================================



:Type:	Integer	
:Range:	{0 .. 1000000}	
:Default:	10	



This option determines the number of old merit function values that are stored by PATH.



**Remark** 

The setting of this option should only be changed by specialist users.
