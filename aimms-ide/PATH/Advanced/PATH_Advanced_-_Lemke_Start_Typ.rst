Lemke Start Type
================



:Type:	Selection	
:Range:	The settings listed below	
:Default:	Slack	



The type of Lemke starts. Possible values are:



*	Slack
*	Advanced
*	Restart stationary limit




**Remark** 


The setting of this option should only be changed by specialist users.
