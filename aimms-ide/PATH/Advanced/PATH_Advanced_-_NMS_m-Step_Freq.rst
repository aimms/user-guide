NMS m-Step Frequency
====================



:Type:	Integer	
:Range:	{0 .. 1000000}	
:Default:	10	



This option determines the frequency at which so-called 'm-steps' are taken.



**Remark** 

The setting of this option should only be changed by specialist users.
