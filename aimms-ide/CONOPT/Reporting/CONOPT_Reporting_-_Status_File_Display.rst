Status File Display
===================



:Type:	Selection	
:Range:	The settings listed below	
:Default:	None	



This option determines the amount of information that is printed to the status file 'conopt.sta'. Possible values are:



*	None
*	Summary
*	Iteration log
*	Detailed iteration log
