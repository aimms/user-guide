Primal pivot
============



:Type:	Selection	
:Range:	The settings listed below	
:Default:	Automatic	



This option specifies the primal pivot selection algorithm. Possible values are:



    *	Automatic
    *	Exact
    *	Partial
    *	Steepest
    *	Change
    *	Sprint
