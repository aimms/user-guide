Output Level
============



:Type: 	Integer
:Range: 	{-1 .. 63}
:Default: 	1



The option sets the level of detail in (LP) solver output. If 0 then there should be no output in normal circumstances. 1 is probably the best value for most uses, while 2 and 3 give more information.
