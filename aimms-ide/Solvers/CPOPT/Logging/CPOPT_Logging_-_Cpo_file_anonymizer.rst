Cpo File Anonymizer
===================



:Type:	Selection	
:Range:	The settings listed below	
:Default:	Off	



This option can be used to anonymize the names of variables and constraints in the cpo file, as generated by switching on the option **Cpo File**. Possible values are:



    *	Off
    *	On
