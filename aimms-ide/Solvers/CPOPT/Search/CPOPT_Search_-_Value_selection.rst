Value Selection
===============



:Type:	Selection	
:Range:	The settings listed below	
:Default:	Automatic	



This option specifies the value selection heuristic being used in a search strategy. Possible values are:



    *	Automatic
    *	Smallest impact
    *	Largest impact
    *	Smallest success rate
    *	Largest success rate
    *	Smallest value
    *	Largest value
