Simplex Thread Limit
====================



:Type:	Integer	
:Range:	{-1 .. 128}	
:Default:	-1	



This option specifies the number of threads used by the dual simplex algorithm. At the default value of -1 the thread count is determined by the option **Thread Limit**.
