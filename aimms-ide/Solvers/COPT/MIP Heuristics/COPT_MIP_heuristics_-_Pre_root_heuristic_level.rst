Pre Root Heuristic Level
========================



:Type:	Selection
:Range:	The settings listed below
:Default:	Automatic



This option controls the level of pre-root heuristics applied before solving the root node relaxation. Possible values are:

    *	Automatic
    *	Off
    *	Fast
    *	Normal
    *	Aggressive



**Note**

*	This option was added in COPT 8.0.
