.. _option-COPT-pool_size:


Pool Size
=========



:Type:	Integer	
:Range:	{0 .. 2000000000}	
:Default:	0	



This option determines how many MIP solutions are stored in the :doc:`COPT_Solution_Pool <../COPT_Solution_Pool>` .



**Note** 

*	This option only affects mixed integer programming (MIP) models.




**Learn more about** 

*	:doc:`COPT_Solution_Pool <../COPT_Solution_Pool>`
