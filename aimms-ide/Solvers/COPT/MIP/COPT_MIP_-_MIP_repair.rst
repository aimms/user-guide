MIP repair
==========



:Type:	Selection
:Range:	The settings listed below
:Default:	Only when time left



This option controls the level of effort applied to repairing the MIP solution in case of numerical issues. Settings other than 'Off' extend the time limit to allow COPT to attempt a repair. Possible values are:

    *	Only when time left
    *	Off
    *	Fast
    *	Normal
    *	Aggressive



**Note**

*	This option was added in COPT 8.0.
