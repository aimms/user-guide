.. _option-ODHCPLEX-presolve:


Presolve
========



:Type:	Selection	
:Range:	The settings listed below	
:Default:	Automatic	



This option determines whether the CPLEX presolve should be applied to the full model. Possible values are:



    *	Off
    *	Automatic
    *	On
