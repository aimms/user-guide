MIP Heuristic Local Search
==========================



:Type:	Selection	
:Range:	The settings listed below	
:Default:	Automatic	



This option specifies whether or not to enable the MIP local search heuristic. Possible values are:



    *	Automatic
    *	Off
    *	On




Setting 'Automatic' lets Knitro determine automatically from :doc:`KNITRO_MIP_-_MIP_Heuristic_Strategy <KNITRO_MIP_-_MIP_Heuristic_Strategy>` . Setting 'Off' means that local search heuristic is not applied.
