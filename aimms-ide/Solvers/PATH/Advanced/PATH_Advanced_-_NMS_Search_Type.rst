NMS Search Type
===============



:Type:	Selection	
:Range:	The settings listed below	
:Default:	Line	



This option determines which search type is used by the NMS path search method. Possible values are:



    *	Arc
    *	Line




**Remark** 


The setting of this option should only be changed by specialist users.
