Calling Aimms
=============



This chapter discusses the command line options of the AIMMS program,
and explains the details for running AIMMS end-user applications. In
addition, the chapter explains how you can link AIMMS to your own
program as a DLL, and presents a short overview of the functionality
available through the AIMMS- specific Application Programming Interface
(API) provided by this DLL.
