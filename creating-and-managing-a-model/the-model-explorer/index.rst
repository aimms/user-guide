The Model Explorer
==================



This chapter introduces the interactive **Model Explorer** that is part
of the AIMMS system. With the **Model Explorer** you have easy access to
every component of the source of your model. In this chapter, you are
introduced to the model tree, and you are shown which model information
can be added to the model tree. In addition, the basic principles of
working with the **Model Explorer** are explained.
